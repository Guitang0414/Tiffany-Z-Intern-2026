## 1. 登录与后台概览

### 登录方式

1. 浏览器打开 `www.epochtimesnw.com/wp-admin`
2. 输入用户名和密码，点击「登入」
3. 登录后进入 WordPress 控制台

### 后台常用入口（左侧菜单）

| 位置 | 用途 |
| :--- | :--- |
| 文章 → 全部文章 | 查看和管理已有文章 |
| 文章 → 新增文章 | 创建新文章 |
| 媒體 → 媒體庫 | 管理已上传的图片和文件 |
| 文章 → 分類 | 管理文章分类 |
| 文章 → 標籤 | 管理文章标签 |

---

## 2. 创建新文章

1. 点击左侧菜单 **「文章」→「新增文章」**
2. 在页面顶部大输入框中输入**文章标题**
3. 在标题下方的编辑区域撰写正文
4. 在右侧和下方设置分类、标签、特色图片等
5. 完成后点击右侧「發佈」区域的 **「發佈」** 按钮

## 3. 编辑器界面介绍

传统编辑器的界面类似 Word，顶部有两排工具栏按钮。

### 编辑区上方的两个标签

| 标签 | 用途 |
| :--- | :--- |
| **預覽**（左边） | 所见即所得模式，像 Word 一样直接编辑 |
| **程式碼**（右边） | HTML 源码模式，可以直接编辑 HTML（进阶用户使用） |

> 日常使用请保持在 **「預覽」** 标签下操作。

### 第一排工具栏

| 按钮                | 功能                    |
| :---------------- | :-------------------- |
| **段落** 下拉菜单       | 设置文字格式：段落、标题 1-6 等    |
| **B**             | 加粗                    |
| **I**             | 斜体                    |
| 无序列表按钮（三条横线 + 圆点） | 添加带黑点的列表              |
| 有序列表按钮（三条横线 + 数字） | 添加带数字的列表              |
| 引用按钮（❝）           | 添加引用样式                |
| 对齐按钮（四个）          | 左对齐 / 居中 / 右对齐 / 两端对齐 |
| 链接按钮（🔗）          | 插入/编辑超链接              |
| 分隔线按钮             | 插入水平分隔线               |

### 第二排工具栏

| 按钮 | 功能 |
| :--- | :--- |
| **ABC** (删除线) | 给文字加删除线 |
| **A** (文字颜色) | 修改文字颜色 |

### 编辑区上方按钮

| 按钮 | 功能 |
| :--- | :--- |
| **新增媒體** | 插入图片、视频等媒体文件 |

## 4. 文字排版

### 设置标题

1. 选中要设为标题的文字
2. 点击工具栏左侧的 **「段落」下拉菜单**
3. 选择需要的标题级别：
   - **标题 2** — 文章内的大标题
   - **标题 3** — 文章内的小标题
   - **段落** — 普通正文

### 加粗和斜体

- 选中文字 → 点击 **B** 加粗
- 选中文字 → 点击 **I** 斜体
- 或使用快捷键 **Ctrl+B** / **Ctrl+I**（Mac: **Cmd+B** / **Cmd+I**）

### 插入链接

1. 选中要加链接的文字
2. 点击工具栏的 **链接按钮（🔗）**，或按 **Ctrl+K**（Mac: **Cmd+K**）
3. 粘贴 URL 地址
4. 按 Enter 确认

### 换行 vs 换段

- **Enter** — 新起一段（段落之间有较大间距）
- **Shift + Enter** — 同一段内换行（行距较小）

## 5. 图片操作

### 在文章中插入图片

1. 将光标放在正文中要插入图片的位置
2. 点击编辑区上方的 **「新增媒體」** 按钮
3. 弹出媒体窗口：
   - **上傳檔案** 标签 — 从电脑拖拽或选择图片上传
   - **媒體庫** 标签 — 从已上传的图片中选择
4. 选中图片后，在右侧可以设置：
   - **说明（Caption）** — 图片下方的图说文字
   - **替代文字（Alt Text）** — 图片的文字描述（SEO 用）
   - **对齐方式** — 无 / 左 / 居中 / 右
   - **尺寸** — 缩略图 / 中 / 大 / 完整尺寸
5. 点击 **「插入至文章」** 完成

### 给图片添加图说

在插入图片时，媒体窗口右侧有一个 **「说明（Caption）」** 字段，在这里输入文字，图片插入后下方就会自动显示图说。

如果图片已经插入了，想补加图说：
1. 点击文章中的图片
2. 图片上方会出现一个小工具栏，点击 **铅笔图标（编辑）**
3. 在弹出的编辑窗口中填写「说明」字段
4. 点击「更新」

### 调整已插入图片的大小

1. 点击文章中的图片
2. 图片四角会出现小方块拖拽点，拖动即可调整大小
3. 或者点击铅笔图标进入编辑，在「尺寸」选项中精确设置

## 6. 特色图片（置顶图片）

特色图片是文章的「封面图」，显示在文章顶部和列表缩略图中。

### 设置特色图片

1. 在编辑文章页面，向下滚动右侧栏
2. 找到 **「特色圖片」** 区域（通常在分类、标签下方）
3. 点击 **「設定特色圖片」**
4. 在弹出的媒体窗口中上传或选择图片
5. 点击 **「設定特色圖片」** 确认

### 更换特色图片（不需要删除文章！）

1. 在右侧栏找到「特色圖片」区域，这里会显示当前的封面图缩略图
2. 点击缩略图下方的 **「移除特色圖片」**
3. 图片被移除后，重新点击 **「設定特色圖片」** 选一张新的
4. 点击页面上的 **「更新」** 保存

> 就这么简单，不需要把文章扔进垃圾桶！

### 在特色图片下方添加图说

特色图片**没有自带的图说输入框**。如果需要在封面图下方显示图说：

**方法一：利用媒体库的「说明」字段**
1. 点击「設定特色圖片」打开媒体窗口
2. 选中图片后，在右侧找到 **「说明（Caption）」** 字段，输入图说
3. 确认设置
4. 是否显示取决于主题支持，请用「预览」检查效果

**方法二：在正文开头手动添加（推荐，最可控）**
1. 在文章正文的最开头输入图说文字，如"图：xxxxx（来源：Getty Images）"
2. 选中文字，设为**斜体**，与正文区分
3. 这种方式不受主题限制，一定能显示

## 7. 列表与黑点（项目符号）

### 创建带黑点的列表

**方法一：使用工具栏按钮**
1. 将光标放在要开始列表的位置
2. 点击工具栏的 **无序列表按钮**（三条横线旁边有圆点的图标）
3. 输入第一项内容，按 **Enter** 自动添加下一项
4. 列表结束时，连按两次 **Enter** 退出列表

**方法二：选中已有文字转为列表**
1. 选中已经写好的多行文字
2. 点击工具栏的 **无序列表按钮**
3. 每一行会自动变成一个列表项

### 解决列表行距太密的问题

传统编辑器中，列表项的行距由主题 CSS 控制，可能和普通段落不一致。

**方法一：在列表项之间加空行**
1. 在一个列表项末尾按 **Shift + Enter** 插入空行
2. 再按 **Enter** 开始下一项
3. 这样可以增加列表项之间的间距

**方法二：切换到「程式碼」模式手动调整**
1. 点击编辑区右上角的 **「程式碼」** 标签
2. 找到列表的 HTML 代码，类似：
   ```html
   <ul>
   <li>第一项</li>
   <li>第二项</li>
   </ul>
   ```
3. 在 `<li>` 标签中添加行距样式：
   ```html
   <ul style="line-height: 2;">
   <li>第一项</li>
   <li>第二项</li>
   </ul>
   ```
4. 切换回「預覽」模式查看效果

**方法三：不用列表功能，手动加黑点（最简单）**
1. 不点列表按钮，就用普通文字
2. 每行前面直接输入黑点符号 `●`（复制粘贴这个符号就行）
3. 这样行距和普通正文**完全一致**

> **输入黑点符号：** 直接复制 `●` 或 `•`；Mac 下也可按 **Option + 8** 输入 `•`

## 8. 分类、标签与 SEO

### 设置分类

1. 编辑页面右侧栏 → 找到 **「分類」** 区域
2. 勾选文章所属的分类（可多选）
3. 如果需要新分类，点击「+ 新增分類」

### 设置标签

1. 编辑页面右侧栏 → 找到 **「標籤」** 区域
2. 输入关键词，点击「新增」或按 Enter
3. 建议每篇文章 3-5 个标签

### Rank Math SEO

你们的站点安装了 **Rank Math SEO** 插件，编辑区下方有 Rank Math 面板：
- 会显示 **SEO 评分**（如 "SEO: 5/100"）
- 可以设置 Focus Keyword（聚焦关键词）
- 根据提示优化标题、描述和内容

基本 SEO 建议：
- 标题包含核心关键词
- 文章开头第一段简明概述全文要点
- 插入图片时填写「替代文字（Alt Text）」

## 9. 保存、预览与发布

右侧栏顶部的 **「發佈」** 区域包含所有发布相关操作：

| 按钮 | 功能 |
| :--- | :--- |
| **儲存草稿** | 保存但不发布，只有自己能看到 |
| **預覽** | 在新标签页中查看发布后的效果 |
| **發佈** | 正式发布，所有人可见 |

### 发布流程

1. 确认内容、分类、标签、特色图片都已设置
2. 点击 **「預覽」** 检查排版效果
3. 确认无误后，点击 **「發佈」**

### 修改发布状态和时间

在「發佈」区域中：
- **状态：草稿** → 点击「编辑」可改为待审核等状态
- **可见度：公开** → 点击「编辑」可改为私密或密码保护
- **立即发布** → 点击「编辑」可设置定时发布的日期和时间

## 10. 修改已发布的文章

1. 进入 **「文章」→「全部文章」**
2. 找到要修改的文章，鼠标移到文章标题上
3. 点击出现的 **「编辑」** 链接
4. 修改内容
5. 点击右侧的 **「更新」** 按钮保存（修改立即生效）

### 快速编辑

在「全部文章」列表中，鼠标移到文章标题上，点击 **「快速编辑」** 可以快速修改标题、分类、标签、状态等，不需要进入完整编辑页面。

## 11. 修订历史与版本记录

WordPress 内置了**修订版本（Revisions）**功能。每次点击「儲存草稿」或「更新」都会自动保存一个历史版本。

### 查看修订历史

1. 打开要查看的文章，进入编辑页面
2. 在编辑区下方，找到 **「修訂版本」** 区域（需要向下滚动）
3. 点击 **「瀏覽」** 进入版本对比页面
4. 页面顶部有一个**时间滑块**，可以拖动查看不同时间点的版本
5. 页面会用**红色（删除）**和**绿色（新增）**显示每个版本的改动
6. 如果想恢复到某个旧版本，点击 **「還原此修訂版本」**

### 注意事项

- 每次保存都会产生一个修订版本，不用担心内容丢失
- 草稿和已发布的文章都有修订历史
- **自动保存：** WordPress 每隔约 60 秒会自动保存一次，即使你没有手动保存
- 恢复旧版本后，当前版本不会丢失（它也被记录在历史中）

## 12. 常见问题 Q&A

> 以下问题来自编辑团队的实际反馈。

---

### Q1: 如何更换置顶图片？不想把文章删掉重新建。

完全不需要删除文章！

1. 打开文章编辑页面
2. 在右侧栏找到 **「特色圖片」** 区域
3. 点击当前图片下方的 **「移除特色圖片」**
4. 移除后，重新点 **「設定特色圖片」** 选新图
5. 点击 **「更新」** 保存

文章内容完全不受影响。详见[第 6 节](#6-特色图片置顶图片)。

---

### Q2: 如何在文字左边加黑点？加了之后行距太密，和正文不一样，怎么办？

**加黑点：** 点击工具栏的 **无序列表按钮**（圆点 + 横线的图标）即可。

**行距太密的问题：** 这是主题样式造成的。三种解决方案：

1. **列表项之间加空行** — 在列表项末尾按 Shift+Enter 插入空行
2. **切换到「程式碼」模式** — 给 `<ul>` 标签添加 `style="line-height: 2;"`
3. **最简单：不用列表功能，手动加黑点** — 直接在每行前面复制粘贴 `●` 符号，行距和正文完全一致

详见[第 7 节](#7-列表与黑点项目符号)。

---

### Q3: 如何在置顶图片（特色图片）下面加图说？

特色图片**没有自带的图说输入框**。

**推荐做法：** 在文章正文的最开头输入一行图说文字（如"图：xxxxx（来源：Getty Images）"），选中后设为**斜体**，让它看起来像图说。这个方法最可控，一定能显示。

详见[第 6 节](#6-特色图片置顶图片)。

---

### Q4: 一敲键盘，东西就丢失了，怎么回事？

**先别慌，立刻按 Ctrl+Z（Mac: Cmd+Z）撤销！** 可以连续多次撤销。

常见原因：

1. **选中了一段文字后按了字母键** — 选中状态下按任何字母会替换掉选中的内容。预防：编辑前确认光标在正确位置闪烁，没有误选大段文字
2. **按了 Backspace/Delete 误删** — 直接 Ctrl+Z 撤销
3. **长时间没保存，浏览器崩溃** — 养成随时 **Ctrl+S** 保存的习惯。好消息：WordPress 每 60 秒会自动保存一次，可以在[修订历史](#11-修订历史与版本记录)中找回

---

### Q5: 之前保存的草稿 / 历史版本在哪里找？

WordPress 会自动保存每一次修改的历史。

- 打开文章编辑页面 → 向下滚动 → 找到 **「修訂版本」** 区域 → 点击「瀏覽」
- 可以对比任意两个版本的差异，也可以一键恢复到旧版本
- 草稿和已发布的文章都有历史记录

详见[第 11 节](#11-修订历史与版本记录)。

---

### Q6: 编辑器里看到的排版和发布后不一样？

正常现象。编辑器的样式和网站前台的主题样式不完全一致。**发布前请务必点「預覽」检查最终效果。**

---

### Q7: 图片上传失败？

- 检查图片文件大小，建议控制在 **2MB 以内**
- 支持的格式：JPG、PNG、GIF、WebP
- 如果反复失败，尝试刷新页面后重新上传

---

### Q8: 从 Word 或网页复制文字过来，格式很乱？

从外部复制的文字会带有原来的 HTML 格式，导致排版混乱。

**方法一：使用「粘贴为纯文字」按钮**
1. 点击第二排工具栏的 **「粘贴为纯文字」** 按钮（剪贴板 + T 图标），使其高亮激活
2. 然后粘贴内容，所有格式会被清除，只保留纯文字
3. 粘贴完后再手动设置需要的格式

**方法二：先粘贴到记事本**
1. 先把内容粘贴到「记事本」（Windows）或「文本编辑」（Mac，纯文本模式）
2. 再从记事本复制到 WordPress
